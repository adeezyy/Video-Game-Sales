# Video Game Hit/Flop Predictor

## How to Run

1. Clone this repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Dataset

Based on `Video_Games_Sales_as_at_22_Dec_2016.csv`
